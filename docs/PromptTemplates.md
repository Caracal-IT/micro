# Prompt Templates You Can Use Today

Organized by use case: with copy-paste-ready prompt scaffolds.

## Writing & Content Creation
“Write a [tone] blog post about [topic] in [#] paragraphs.”

“Rewrite this in the style of [brand/person].”

“Create 3 catchy headlines for [product or theme].”

## Data Analysis
“Summarize the key trends in this CSV data.”

“Explain anomalies in the data and suggest hypotheses.”

“Convert this raw data into an insight-rich executive summary.”

## Learning & Summarizing
“Teach me [concept] like I’m 12.”

“Summarize this paper with key takeaways, 3 bullet points max.”

“What are the pros and cons of [approach] in plain English?”

## Emails & Messaging
“Write a [friendly/formal] follow-up email about [topic].”

“Summarize this long email into 2 sentences + a subject line.”

## Developer Prompts
“Write a Python function that does [task] with comments.”

“Debug this code step-by-step and explain the issue.”

“Convert this script into a REST API in Flask.”
